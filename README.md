# Facebook Web Application QA Test Project

This project demonstrates manual QA testing performed on the Facebook web application as part of a QA portfolio.

## Environment
- Browser: Google Chrome
- OS: Windows 11
- Application: Facebook Web

## Scope of Testing
- Login and logout
- News Feed interactions
- Posting status updates
- Like, comment, share features
- Profile editing
- Friend request flow
- Messenger access
- Notifications

## Artifacts Included
- Test Plan
- Test Cases
- Bug Reports
- Exploratory Testing Notes
